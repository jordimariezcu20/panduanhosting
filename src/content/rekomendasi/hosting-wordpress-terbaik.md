WordPress adalah CMS yang membutuhkan hosting yang tepat untuk berfungsi optimal. Hosting yang salah bisa membuat WordPress lambat, sering down, atau sulit dikonfigurasi. ## Kriteria Evaluasi

Kami menilai berdasarkan:
- **Kecepatan** — Time to First Byte (TTFB) dan loading time WordPress
- **Uptime** — ketersediaan server sepanjang waktu
- **WordPress-specific features** — auto-installer, staging, WordPress toolkit
- **Support** — responsivitas dan keahlian mengatasi masalah WordPress
- **Harga** — nilai yang didapat vs biaya yang dikeluarkan

## Tips Memilih Hosting WordPress

1. **Pertimbangkan lokasi pengunjung** — jika target pengunjung dari Indonesia, server lokal penting
2. **Cek apakah ada LiteSpeed** — jauh lebih cepat dari Apache untuk WordPress

4. **Test support** — coba live chat sebelum memutuskan, cek responsivitas dan keahlian
5. **Perhatikan harga renewal** — harga promo bisa sangat murah, tapi cek harga tahun kedua

## FAQ: Hosting WordPress

**Apakah hosting biasa bisa untuk WordPress?**
Ya, tapi hosting yang dioptimalkan untuk WordPress memberikan performa lebih baik. Yang terpenting: PHP 7.4+, MySQL, dan bisa install WordPress (Softaculous atau manual).

**Berapa storage yang dibutuhkan untuk WordPress?**
WordPress dasar hanya 50–100MB. Tapi dengan gambar, plugin, dan konten, bisa berkembang menjadi 1–5GB. Pilih hosting dengan minimal 5GB storage untuk jangka panjang.

**Apakah WordPress hosting dan web hosting berbeda?**
WordPress hosting adalah web hosting yang dioptimalkan khusus untuk WordPress — biasanya punya LiteSpeed cache, WordPress auto-installer, dan mungkin fitur WordPress khusus seperti staging. Dari sisi teknis, keduanya bisa menjalankan WordPress.
