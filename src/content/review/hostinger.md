## Kelebihan Hostinger

### 1. Harga Paling Kompetitif

Ini keunggulan utama Hostinger. Harga paket Premium dan Business Hostinger, bahkan saat renewal, lebih murah dibandingkan banyak kompetitor dengan spesifikasi serupa.

Untuk pemula yang sensitif budget, Hostinger memberikan nilai terbaik di kelasnya.

### 2. hPanel yang Modern dan Intuitif

Berbeda dari cPanel yang terlihat "lawas", hPanel Hostinger dirancang dengan UX modern yang lebih mudah dipahami oleh pengguna baru.

Semua fitur penting mudah ditemukan, dan tampilan visualnya jauh lebih bersih. Banyak pengguna yang baru pertama pakai hosting merasa hPanel lebih mudah dari cPanel.

Kekurangannya: karena bukan standar industri, tutorial cPanel dari internet tidak berlaku langsung di hPanel.

### 3. LiteSpeed + NVMe SSD

Hostinger menggunakan kombinasi LiteSpeed web server dan NVMe SSD — dua teknologi yang berkontribusi signifikan pada kecepatan loading website.

NVMe secara khusus lebih cepat dari SSD SATA standar, dan ini terasa pada performa loading file.

### 4. Server Singapura — Cepat untuk Indonesia

Singapura adalah hub teknologi Asia Tenggara dengan latency rendah ke Indonesia (umumnya 5–20ms dari kota-kota besar Indonesia). Ini menjadikan Hostinger pilihan yang performa-nya sangat baik untuk pengunjung Indonesia.

### 5. Backup Mingguan Gratis

Semua paket Hostinger menyertakan backup mingguan gratis. Paket Business mendapatkan backup harian. Ini value yang baik — banyak hosting mengenakan biaya tambahan untuk fitur ini.

### 6. Banyak Resource Pembelajaran

Hostinger menyediakan blog, tutorial, dan dokumentasi yang sangat lengkap. Untuk pengguna baru, ini sangat membantu untuk belajar mandiri.

### 7. Sertifikat SSL Gratis

Semua paket sudah termasuk SSL gratis (Let's Encrypt) yang diperbarui otomatis.

### 8. Subdomain dan Email Unlimited

Di paket Premium ke atas, kamu bisa membuat subdomain dan akun email sebanyak apapun yang dibutuhkan.

## Kekurangan Hostinger

### 1. Support Bahasa Indonesia Terbatas

Ini kekurangan terbesar Hostinger untuk pengguna Indonesia. Support utama tersedia dalam bahasa Inggris. Ada beberapa artikel panduan dalam bahasa Indonesia, tapi live chat cenderung dalam bahasa Inggris.

Bagi pengguna yang tidak nyaman berkomunikasi dalam bahasa Inggris saat ada masalah teknis, ini bisa menjadi hambatan.

### 2. Uptime Sesekali Berfluktuasi

Meski secara umum uptime Hostinger baik, ada laporan dari beberapa pengguna tentang downtime singkat yang lebih sering dari provider premium lokal.

### 3. Tidak Ada Server di Indonesia

Server Hostinger terdekat ada di Singapura. Meski latency cukup baik untuk Indonesia, tidak seoptimal hosting dengan server fisik di Indonesia.

### 4. Fitur Terbatas di Paket Entry-Level

Paket "Single" Hostinger sangat terbatas — 1 website saja dan resource ketat. Untuk mendapat value yang baik, disarankan minimal paket Premium.
