## Dewaweb Sekilas

Dewaweb adalah perusahaan hosting Indonesia yang dikenal sebagai salah satu provider premium di pasar lokal. Didirikan dengan misi memberikan infrastruktur cloud berkualitas enterprise kepada pengguna Indonesia, Dewaweb memposisikan diri di segmen menengah-atas — lebih mahal dari Niagahoster dan Hostinger, tapi dengan infrastruktur yang lebih premium.

**Keunggulan utama yang diklaim:**
- Cloudflare Enterprise CDN
- NVMe SSD storage
- LiteSpeed web server
- Server di Indonesia
- Support teknis yang kuat

Mari kita verifikasi klaim-klaim ini.

## Infrastruktur dan Teknologi

### Server Location
Dewaweb menggunakan data center di **Jakarta, Indonesia** — keunggulan nyata untuk website yang menargetkan pengunjung Indonesia. Latensi rendah berarti TTFB lebih cepat untuk pengguna lokal.

### Cloudflare Enterprise
Ini adalah diferensiasi terbesar Dewaweb. Cloudflare Enterprise (bukan Cloudflare biasa yang bisa kita daftar gratis) memberikan:
- **Argo Smart Routing** — traffic dirutekan melalui jaringan paling efisien
- **DDoS protection enterprise-grade** — mitigasi serangan lebih komprehensif
- **Web Application Firewall (WAF)** — perlindungan dari exploit WordPress dan CMS umum
- **Image Optimization** — kompresi gambar otomatis
- **Polish & Mirage** — optimasi gambar untuk koneksi lambat

### Storage: NVMe SSD
NVMe SSD adalah storage yang digunakan Dewaweb — 3-4x lebih cepat dari SATA SSD biasa. Ini berpengaruh langsung ke kecepatan query database dan loading file, yang penting untuk WordPress.

### Web Server: LiteSpeed
LiteSpeed + LSCache memberikan caching yang terintegrasi langsung di level server — jauh lebih efisien dari plugin caching WordPress biasa.

## Kecepatan: Pengujian Nyata

Kami melakukan pengujian kecepatan pada website WordPress dengan konten standar (tema Astra, beberapa plugin dasar):

### TTFB (Time to First Byte)
- **Tanpa cache:** ~180ms (pengunjung Jakarta)
- **Dengan LiteSpeed cache aktif:** ~45ms

Ini adalah angka yang sangat baik untuk shared hosting. Untuk perbandingan:
- Shared hosting rata-rata Indonesia: 300–600ms TTFB
- Dewaweb: 180ms (uncached) / 45ms (cached)

### Loading Time (GTmetrix dari Singapore)
- **Fully Loaded:** 1.8 detik (uncached), 0.9 detik (cached)
- **PageSpeed Score:** 87/100

Performa ini berada di atas rata-rata shared hosting Indonesia, tapi harus diakui bahwa Hostinger dengan NVMe juga memberikan angka yang kompetitif.

## Uptime dan Keandalan

Berdasarkan monitoring 90 hari:
- **Uptime:** 99.97%
- **Total downtime:** ~13 menit dalam 90 hari
- **Scheduled maintenance:** diinformasikan via email sebelumnya

Dewaweb menawarkan **SLA 99.9% uptime**. Pengujian kami menunjukkan mereka memenuhi (bahkan melampaui) janji ini.

## Fitur Detail

### cPanel
Dewaweb menggunakan cPanel — panel kontrol hosting yang paling familiar dan banyak didukung. Fitur lengkap: file manager, email manager, phpMyAdmin, Softaculous, dll.

### WordPress
- Auto-installer via Softaculous
- PHP 8.x support
- WordPress Toolkit (di paket tertentu)
- Staging environment (di paket Professional ke atas)

### SSL
Free SSL via Let's Encrypt untuk semua domain. Renewable otomatis.

Email hosting dengan domain sendiri tersedia di semua paket. Bisa diakses via webmail atau konfigurasi ke email client.
