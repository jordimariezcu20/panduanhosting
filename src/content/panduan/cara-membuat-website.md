Membuat website sendiri tidak sesulit yang dibayangkan. Dengan platform modern seperti WordPress, kamu bisa punya website profesional yang berfungsi penuh tanpa menulis satu baris kode pun. Panduan ini membahas **cara membuat website** dari nol untuk pemula.

## Berapa Lama dan Berapa Biaya Membuat Website?

Sebelum mulai, ini gambaran realistis:

**Waktu:**
- Setup awal (hosting + domain + WordPress): **30–60 menit**
- Pilih dan kustomisasi tema: **1–3 jam**
- Buat halaman dasar (Beranda, Tentang, Kontak): **2–4 jam**
- Total hingga website siap dipublikasi: **setengah hari**

**Biaya (per tahun):**
- Domain (.com atau .id): Rp 150.000–200.000
- Hosting (shared): Rp 150.000–600.000
- **Total: Rp 300.000–800.000/tahun**

Banyak hosting menawarkan domain gratis untuk tahun pertama, jadi bisa lebih hemat.

## Langkah 1: Tentukan Tujuan Website

Pertanyaan paling pertama yang harus dijawab: **website ini untuk apa?**

Tujuan website menentukan platform dan fitur yang dibutuhkan:

| Tujuan | Platform yang Cocok |
|---|---|
| Blog personal / konten | WordPress (self-hosted) |
| Portfolio kreatif | WordPress, Webflow, Squarespace |
| Website bisnis informasional | WordPress |
| Toko online | WordPress + WooCommerce, Shopify |
| Website sekolah/organisasi | WordPress, Google Sites |
| Landing page sederhana | WordPress, Google Sites |

Untuk hampir semua keperluan, **WordPress self-hosted** adalah pilihan paling fleksibel dan cost-effective.

## Langkah 2: Pilih dan Beli Domain

Domain adalah alamat website kamu (contoh: `tokobaju.com`, `portofolio-andi.id`).

**Tips memilih nama domain:**
- Singkat, mudah dieja, mudah diingat
- Relevan dengan bisnis atau nama kamu
- Hindari angka dan tanda hubung berlebihan
- Gunakan ekstensi `.com` untuk jangkauan global atau `.id` untuk identitas Indonesia

## Langkah 3: Beli Hosting

Hosting adalah tempat menyimpan file website kamu di internet. Tanpa hosting, website tidak bisa diakses.

**Untuk pemula, pilih shared hosting** karena:

- Tidak perlu keahlian teknis
- Sudah termasuk cPanel, SSL gratis, email bisnis

## Langkah 4: Install WordPress

Setelah hosting aktif (biasanya dalam 30 menit–1 jam setelah pembayaran):

1. Login ke **cPanel** (link ada di email dari hosting)
2. Cari **"Softaculous"** atau **"WordPress"** di bagian Software
3. Klik **"Install"**
4. Isi:
   - Protocol: `https://`
   - Domain: pilih domain kamu
   - Directory: kosongkan
   - Site Name: nama website
   - Admin Username dan Password (simpan baik-baik!)
   - Admin Email
5. Klik **"Install"**

WordPress siap dalam 2–3 menit. Akses admin di `namadomain.com/wp-admin`.

## Langkah 5: Pilih Tema WordPress

Tema menentukan tampilan visual website. Di WordPress Admin:

1. Masuk ke **Tampilan → Tema → Tambah Baru**
2. Cari tema yang sesuai dengan tujuan website
3. Klik **"Preview"** untuk melihat tampilan
4. Klik **"Install"** lalu **"Aktifkan"**

**Tema gratis terbaik untuk pemula:**

- **Astra** — sangat populer, ringan, banyak template starter gratis
- **GeneratePress** — super cepat, minimalis, SEO-friendly
- **Neve** — modern, mobile-first, banyak pilihan template

Kamu bisa mengganti tema kapan saja tanpa kehilangan konten.

## Langkah 6: Kustomisasi Tampilan

Kebanyakan tema modern mendukung **Elementor** atau **Gutenberg** sebagai page builder visual — kamu bisa edit tampilan dengan drag and drop.

**Dengan Elementor (Plugin gratis):**
1. Install plugin Elementor dari **Plugin → Tambah Baru**
2. Aktifkan Elementor
3. Saat edit halaman, klik **"Edit with Elementor"**
4. Kustomisasi layout, warna, font, dan konten secara visual

**Elemen yang bisa dikustomisasi:**
- Logo dan nama website (di Tampilan → Kustomisasi)
- Warna utama dan warna aksen
- Jenis dan ukuran font
- Header dan footer
- Layout halaman

## Langkah 7: Buat Halaman Penting

Website yang baik minimal memiliki halaman-halaman ini:

**Beranda (Homepage)**
Halaman pertama yang dilihat pengunjung. Harus jelas menjawab: website ini tentang apa dan apa yang bisa didapat pengunjung.

**Tentang Kami / About**
Cerita singkat tentang siapa kamu/bisnis kamu, apa yang dilakukan, dan mengapa pengunjung harus percaya.

**Layanan atau Produk**
Daftar layanan atau produk yang ditawarkan beserta detail dan harga (jika relevan).

**Kontak**
Cara menghubungi kamu: email, nomor WhatsApp, alamat, Google Maps embed, dan formulir kontak.

**Cara membuat halaman di WordPress:**
1. Klik **"Halaman"** di sidebar kiri
2. Klik **"Tambah Baru"**
3. Tulis judul dan konten
4. Klik **"Terbitkan"**

## Langkah 8: Install Plugin Penting

Plugin menambah fitur ke WordPress. Ini plugin dasar yang wajib ada:

| Plugin | Fungsi |
|---|---|
| **Yoast SEO** / **RankMath** | Optimasi SEO, sitemap otomatis |
| **WP Super Cache** | Mempercepat loading website |
| **Wordfence** | Keamanan dan proteksi malware |
| **UpdraftPlus** | Backup otomatis |
| **Contact Form 7** | Formulir kontak |
| **Really Simple SSL** | Memastikan HTTPS aktif di semua halaman |

Cara install: **Plugin → Tambah Baru → Cari nama plugin → Install → Aktifkan**.

## Langkah 9: Atur SEO Dasar

Agar website bisa ditemukan di Google:

**1. Setting Yoast SEO / RankMath**
- Ikuti wizard setup yang muncul setelah aktivasi
- Isi profil website, tipe konten, dan preferensi SEO

**2. Atur Permalink**
Di **Pengaturan → Tautan Permanen → Pilih "Nama Postingan"** → Simpan. Ini menghasilkan URL yang bersih dan SEO-friendly.

**3. Daftarkan ke Google Search Console**
- Buka [search.google.com/search-console](https://search.google.com/search-console)
- Tambahkan property (domain atau URL prefix)
- Verifikasi kepemilikan
- Submit sitemap: biasanya `namadomain.com/sitemap.xml`

Setelah didaftarkan, Google akan mulai mengindeks website kamu.

## Langkah 10: Terbitkan Konten

Website tanpa konten tidak akan punya pengunjung. Langkah paling penting setelah setup adalah membuat konten:

- **Blog/artikel** — untuk website konten, minimal 10–20 artikel awal
- **Deskripsi layanan** — detail, manfaat, dan mengapa kamu berbeda
- **Portofolio** — karya atau proyek yang pernah dilakukan
- **Testimonial** — ulasan dari klien atau pelanggan

Konsistensi adalah kunci. Satu artikel per minggu lebih baik dari sepuluh artikel sekaligus lalu berhenti.

## FAQ: Cara Membuat Website

**Apakah perlu bisa coding untuk membuat website?**

Tidak. WordPress dengan page builder seperti Elementor memungkinkan kamu membuat website profesional tanpa coding sama sekali. Coding diperlukan hanya untuk kustomisasi yang sangat spesifik.

**Berapa lama website baru muncul di Google?**

Setelah didaftarkan ke Google Search Console dan sitemap disubmit, butuh 1–4 minggu untuk halaman mulai terindeks. Peringkat di Google membutuhkan waktu lebih lama — biasanya 3–6 bulan untuk artikel baru.

**Bisakah membuat website sendiri tanpa membeli domain?**

Bisa, dengan platform gratis seperti WordPress.com (domain: namamu.wordpress.com) atau Google Sites. Tapi untuk kesan profesional dan bisnis serius, domain sendiri sangat disarankan.

**Apakah bisa ganti platform setelah website jadi?**

Bisa, tapi ada proses migrasi yang perlu dilakukan. Lebih mudah memilih platform yang tepat dari awal daripada migrasi kemudian.

**Website sudah jadi tapi tidak muncul di Google, kenapa?**

Kemungkinan: (1) belum didaftarkan ke Google Search Console, (2) pengaturan WordPress memblokir robot Google (cek di **Pengaturan → Membaca** — pastikan "Halangi mesin pencari" tidak dicentang), atau (3) website terlalu baru dan belum sempat diindeks.
