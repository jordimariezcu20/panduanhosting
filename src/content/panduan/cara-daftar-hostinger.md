Hostinger adalah hosting global dengan harga terjangkau yang sangat populer di Indonesia. Dengan panel modern (hPanel), performa tinggi, dan promo yang sering menarik, Hostinger menjadi pilihan favorit banyak pengguna baru. Artikel ini memandu cara daftar Hostinger Indonesia dari awal hingga hosting aktif.

## Mengapa Pilih Hostinger?

- **Harga kompetitif** — salah satu hosting termurah dengan kualitas baik
- **hPanel yang intuitif** — panel kontrol modern dan mudah digunakan
- **Performa baik** — server LiteSpeed, SSD NVMe storage
- **SSL gratis** untuk semua paket
- **Domain gratis** di paket tertentu
- **Garansi uang kembali 30 hari**
- **Support 24/7** via live chat (bahasa Inggris, tapi responsif)

## Paket Hosting Hostinger

### Web Hosting (Shared)
- **Single** — 1 website, cocok untuk website pertama
- **Premium** — unlimited website, subdomain unlimited
- **Business** — performa lebih tinggi, prioritas support

### WordPress Hosting
Dioptimalkan untuk WordPress dengan fitur WordPress Acceleration.

### Cloud Hosting
Untuk website dengan traffic tinggi yang butuh resource skalabel.

### VPS Hosting
Kontrol penuh server dengan resource dedicated.

## Langkah 1: Buka Website Hostinger

1. Kunjungi **[hostinger.co.id](/go/hostinger)**
2. Klik **Hosting** di menu navigasi
3. Pilih jenis hosting (Web Hosting untuk pemula)

**Tips:** Cek juga halaman Promo — Hostinger sering menawarkan diskon 60–80% untuk pembelian pertama.

## Langkah 2: Pilih Paket

1. Bandingkan fitur ketiga paket yang tersedia
2. Untuk website pertama, paket **Premium** biasanya menjadi pilihan terbaik (unlimited website dengan harga yang masih terjangkau)
3. Pilih **periode berlangganan** — semakin panjang, semakin murah per bulan:
   - 1 bulan: harga per bulan paling tinggi
   - 12 bulan: balance antara komitmen dan penghematan
   - 48 bulan: harga per bulan paling murah
4. Klik **Tambah ke Keranjang**

## Langkah 3: Konfigurasi Domain

Setelah memilih paket, isi detail domain:

### Daftarkan Domain Baru
1. Ketik nama domain yang diinginkan
2. Pilih ekstensi: `.com`, `.id`, `.net`, `.org`, dll.
3. Cek ketersediaan
4. Jika paket kamu termasuk domain gratis, domain akan ditambahkan tanpa biaya tambahan

### Gunakan Domain yang Sudah Ada
1. Pilih **"Punya domain? Gunakan yang sudah ada"**
2. Masukkan nama domain kamu
3. Setelah hosting aktif, update nameserver domain ke Hostinger

### Gunakan Subdomain Hostinger (Sementara)
Untuk testing atau jika belum siap beli domain, bisa gunakan subdomain gratis dari Hostinger:
- Format: `namamu.hostinger-free.com`

## Langkah 4: Review Pesanan

Halaman checkout menampilkan:
- Paket yang dipilih dan periode
- Domain (jika ada)
- Subtotal dan total

Periksa semua detail. Kamu bisa menerapkan **kode promo** jika punya — masukkan di kolom yang tersedia.

## Langkah 5: Buat Akun atau Login

Hostinger meminta kamu membuat akun atau login:

**Cara daftar akun baru:**
- **Email + Password**: isi email aktif dan buat password
- **Google**: daftar dengan akun Google (lebih cepat)
- **GitHub**: untuk developer

Masukkan data yang diperlukan dan klik **Daftar & Bayar** atau **Checkout**.

## Langkah 6: Pilih Metode Pembayaran

Hostinger Indonesia menerima:

**Kartu Kredit/Debit:**
- Visa, Mastercard
- Proses otomatis

**PayPal:**
- Cocok jika punya akun PayPal

**Transfer Bank / Virtual Account:**
- BCA, Mandiri, BNI, BRI, Permata
- Untuk beberapa bank, konfirmasi mungkin perlu 1–4 jam

**E-wallet:**
- GoPay, OVO, DANA, ShopeePay
- Proses cepat

**Alfamart/Indomaret:**
- Bayar tunai di minimarket

Pilih metode pembayaran dan klik **Bayar Sekarang**.

## Langkah 7: Verifikasi Email

1. Cek inbox email yang digunakan saat daftar
2. Buka email verifikasi dari Hostinger
3. Klik **Verifikasi Email**
4. Kamu diarahkan ke dashboard Hostinger

## Langkah 8: Setup Hosting di hPanel

Setelah akun aktif dan email diverifikasi:

1. Login ke **hpanel.hostinger.com** atau klik link dari email konfirmasi
2. Kamu akan melihat dashboard dengan hosting yang sudah aktif
3. Klik **Kelola** pada hosting kamu

### Antarmuka hPanel

hPanel Hostinger berbeda dari cPanel tradisional — lebih modern dan terorganisir:
- **Website** — kelola file, domain, SSL
- **Email** — buat email domain
- **Database** — MySQL dan phpMyAdmin
- **Advanced** — pengaturan server, PHP version, dll.

## Langkah 9: Install WordPress (Opsional)

Jika ingin install WordPress:
1. Di hPanel → klik **Website**
2. Scroll ke bagian **Auto Installer**
3. Klik **WordPress**
4. Isi detail:
   - URL website (domain kamu)
   - Judul website
   - Username admin
   - Password admin
   - Email admin
5. Klik **Install**
6. Selesai dalam 2–3 menit

## Cara Update Nameserver untuk Domain Luar

Jika domain kamu di GoDaddy, Namecheap, Domainesia, dll:

1. Login ke registrar domain
2. Temukan pengaturan **Nameserver**
3. Ganti dengan nameserver Hostinger:
   - `ns1.dns-parking.com`
   - `ns2.dns-parking.com`
   *(Nameserver Hostinger bisa berbeda — cek di hPanel → Domain → Nameservers untuk server yang tepat)*
4. Simpan perubahan
5. Tunggu propagasi 1–24 jam

## FAQ: Daftar Hostinger

**Apakah Hostinger ada yang gratis?**
Hostinger tidak menawarkan paket hosting gratis permanen. Ada uji coba 30 hari dengan garansi uang kembali jika tidak puas.

**Berapa lama proses aktivasi?**
Hosting biasanya aktif dalam beberapa menit setelah pembayaran dikonfirmasi. Email konfirmasi dikirim segera.

**Bagaimana jika lupa password hPanel?**
Klik "Lupa Password" di halaman login hPanel → masukkan email → cek email untuk link reset password.

**Apakah ada biaya setup?**
Tidak ada biaya setup. Kamu hanya membayar harga paket hosting yang dipilih.

**Bisa tidak bayar dengan transfer BCA?**
Ya, Hostinger menerima transfer BCA melalui Virtual Account BCA. Konfirmasi otomatis setelah transfer berhasil.

**Bagaimana cara menghubungi support Hostinger?**
Via live chat 24/7 di hPanel. Support tersedia dalam bahasa Inggris, tapi ada pilihan bahasa Indonesia juga.
