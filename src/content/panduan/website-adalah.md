**Website** adalah kumpulan halaman web yang saling terhubung, dapat diakses melalui internet menggunakan browser, dan diidentifikasi dengan nama domain tertentu. Setiap website terdiri dari halaman-halaman yang berisi teks, gambar, video, dan elemen interaktif lainnya.

Contoh sederhana: ketika kamu membuka `google.com`, `tokopedia.com`, atau `panduanhosting.id` di browser — itulah website.

## Komponen Utama Sebuah Website

Tiga elemen yang wajib ada untuk sebuah website bisa online:

### 1. Domain
Nama/alamat website yang diketik di browser. Contoh: `tokobaju.com`, `sekolahku.sch.id`. Domain dibeli dari registrar domain dengan biaya sekitar Rp 150.000–250.000 per tahun.

### 2. Hosting
Tempat penyimpanan semua file website di server yang terhubung ke internet 24/7. Tanpa hosting, file website tidak bisa diakses dari luar. Biaya mulai Rp 15.000/bulan.

### 3. Konten
File-file yang membentuk halaman website: HTML (struktur), CSS (tampilan), JavaScript (interaktivitas), gambar, video, dan database (untuk website dinamis).

## Cara Kerja Website

Proses yang terjadi ketika kamu membuka sebuah website:

1. **Kamu mengetik URL** di browser (misal: `panduanhosting.id`)
2. **Browser bertanya ke DNS** — sistem yang menerjemahkan nama domain ke alamat IP server
3. **DNS menjawab** dengan IP server hosting (misal: `103.45.67.89`)
4. **Browser terhubung ke server** dan meminta halaman yang ingin ditampilkan
5. **Server mengirimkan file** HTML, CSS, JavaScript, dan gambar
6. **Browser merender file** tersebut menjadi halaman yang terlihat dan bisa diklik
7. **Halaman tampil** di layarmu dalam hitungan detik

## Jenis-Jenis Website

### Berdasarkan Sifatnya

**Website Statis**
Konten tidak berubah kecuali developer mengubah kode secara manual. Cocok untuk website portfolio atau landing page yang jarang diupdate.

**Website Dinamis**
Konten bisa berubah secara otomatis berdasarkan input pengguna atau data dari database. Sebagian besar website modern bersifat dinamis — termasuk blog, toko online, dan media sosial.

## Perbedaan Website dan Blog

Pertanyaan yang sering muncul:

| | Website | Blog |
|---|---|---|
| **Definisi** | Istilah umum untuk semua halaman web | Jenis website dengan konten berbasis artikel/postingan |
| **Konten** | Statis atau campuran | Terutama artikel dengan tanggal |
| **Update** | Tidak harus rutin | Idealnya rutin (mingguan/harian) |
| **Navigasi** | Menu halaman (Beranda, Tentang, dll) | Daftar postingan terbaru |

Blog **adalah** jenis website. Semua blog adalah website, tapi tidak semua website adalah blog.

## Perbedaan Website dan Aplikasi Web

| | Website | Aplikasi Web |

## Bagaimana Cara Membuat Website?

Ada beberapa pendekatan tergantung kebutuhan dan kemampuan:

### Opsi 1: Platform Website Builder (Paling Mudah)
Wix, Squarespace, Webflow — drag and drop, tidak perlu coding, hosting sudah termasuk. Tapi mahal dan kontrol terbatas.

### Opsi 2: WordPress Self-Hosted (Paling Populer)
Instalasi WordPress di hosting sendiri. Tidak perlu coding, ribuan tema dan plugin tersedia, kontrol penuh. Biaya: Rp 15.000–50.000/bulan untuk hosting + domain.

### Opsi 3: Coding dari Nol (Developer)
HTML, CSS, JavaScript, dan framework seperti React atau Vue. Kontrol total, tapi butuh keahlian teknis tinggi.

Untuk pemula di Indonesia, **WordPress self-hosted** adalah pilihan paling seimbang antara kemudahan, fleksibilitas, dan biaya.

## FAQ: Website

**Apakah website dan web sama?**
"Web" adalah kependekan dari "World Wide Web" — sistem jaringan halaman yang terhubung via internet. Website adalah entitas spesifik yang terdiri dari halaman-halaman di dalam web tersebut.

**Apakah website bisa dibuat gratis?**
Ya, dengan platform seperti Google Sites atau Blogger. Tapi untuk bisnis serius, website dengan domain sendiri dan hosting berbayar jauh lebih profesional dan terpercaya.

**Berapa lama proses pembuatan website?**
Website sederhana dengan WordPress bisa jadi dalam sehari. Website e-commerce kompleks bisa membutuhkan 1–3 bulan.

**Apa perbedaan website dan media sosial?**
Website adalah properti digital yang sepenuhnya kamu miliki dan kendalikan. Media sosial (Instagram, Facebook, TikTok) adalah platform milik perusahaan lain — kamu hanya meminjam "ruang" di sana dan bergantung pada aturan mereka.
